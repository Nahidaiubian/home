Computer Vision Beginner Tutorial

Sixteen weeks, 32 class sessions (two per week): concepts in Class 1, hands-on work in Class 2. Each topic links to a detail page with notes and an embedded PDF viewer. For each session, save your handout as lecture.pdf in the matching folder assets/Tutorial/ComputerVision/WxCy (for example W1C1 for week 1, class 1; W1C2 for week 1, class 2).

Schedule (16 weeks · 32 classes)

Week Class 1 (Concepts) Class 2 (Hands-on / Deep dive)
1 Course intro; what is computer vision; images as data; applications Python setup, Jupyter/OpenCV installation, loading and displaying images
2 Image representation: pixels, channels, grayscale, RGB, image formats Reading, saving, resizing, cropping, and color conversion in OpenCV
3 Basic image operations and NumPy for vision Pixel access, slicing, drawing shapes/text, image arithmetic
4 Image enhancement and filtering basics Blur, Gaussian blur, median filtering, sharpening, noise removal
5 Edges and gradients Sobel, Laplacian, Canny edge detection, parameter tuning
6 Thresholding and binary image processing Global, adaptive, Otsu thresholding; morphological operations
7 Contours, shape analysis, and connected components Finding contours, bounding boxes, area, perimeter, centroid, shape filtering
8 Geometric transformations Translation, rotation, scaling, affine and perspective transforms
9 Feature detection and matching basics Harris corners, ORB/SIFT-style features, descriptor matching
10 Image segmentation fundamentals Region-based segmentation, watershed, color-based segmentation
11 Video processing and motion basics Reading webcam/video, frame processing, background subtraction
12 Object detection fundamentals Face detection, classical detectors, simple tracking in video
13 Introduction to machine learning for vision Feature vectors, train/test split, simple image classification workflow
14 Deep learning for computer vision: CNN basics Using a pretrained CNN for image classification
15 Modern CV tasks: detection, segmentation, transformers Running a pretrained object detector or segmentation model
16 Final beginner projects and review Mini-project demos: edge detector, object tracker, classifier, or vision app
