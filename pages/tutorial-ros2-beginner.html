Robotics With ROS2

Sixteen weeks, 32 class sessions (two per week): concepts in Class 1, hands-on work in Class 2. Each topic links to a detail page with notes and an embedded PDF viewer. For each session, save your handout as lecture.pdf in the matching folder assets/Tutorial/WxCy (for example W1C1 for week 1, class 1; W1C2 for week 1, class 2).

Schedule (16 weeks · 32 classes)

Week Class 1 (Concepts) Class 2 (Hands-on / Deep dive)
1 Course intro, ROS 2 overview, DDS, distributions ROS 2 workspace setup, colcon, packages, nodes, CLI tools
2 ROS 2 communication: topics & QoS Publishers/subscribers in Python/C++, rqt, ros2 topic tools
3 Services, actions, parameters Designing APIs with services/actions, parameter management
4 Launch, composition, lifecycle nodes Launch files in Python, component composition, lifecycle patterns
5 TF2 and time in ROS 2 Frames for mobile bases/manipulators, tf2 tools, bagging and replay
6 URDF/Xacro and robot modeling Building a robot description, RViz2 visualization, basic collision/geometries
7 Gazebo/ignition simulation with ROS 2 World/robot plugins, sim–real interfaces, testing in simulation
8 Sensors and perception pipelines Camera/LiDAR/IMU integration, basic perception node graphs
9 ROS 2 Control and hardware interfaces Controllers, hardware_interface, joint state/command, example arm/base
10 Navigation stack (Nav2) concepts Map server, localization, planners, controllers, BT navigator
11 Nav2 hands-on and tuning Bringing up Nav2 on a differential/omni platform, costmaps, behavior trees
12 Manipulation with MoveIt 2 Motion planning pipelines, planning scenes, pick-and-place demos
13 Multi-robot and distributed systems Namespaces, remapping, discovery, communication patterns
14 Testing, debugging, logging, profiling ros2 test, GTest, launch testing, performance tools
15 Deployment: containers, snaps, real-time & safety Docker, snaps, launch-on-boot, RT kernels, safety patterns
16 Student project presentations System design critique, lessons learned, future extensions
