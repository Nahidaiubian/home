Gazebo Simulation

Sixteen weeks, 32 class sessions (two per week): concepts in Class 1, hands-on work in Class 2. Each topic links to a detail page with notes and an embedded PDF viewer. For each session, save your handout as lecture.pdf in the matching folder assets/Tutorial/Gazebo/WxCy (for example W1C1 for week 1, class 1; W1C2 for week 1, class 2).

Schedule (16 weeks · 32 classes)

Week Class 1 (Concepts) Class 2 (Hands-on / Deep dive)
1 Course intro; role of simulation; Gazebo Classic vs Ignition vs Gazebo Sim Installing Gazebo Sim and ROS 2; running example worlds; GUI tour
2 Gazebo architecture: server, client, worlds, SDF Editing and launching stock worlds; experimenting with physics and lighting
3 SDF and world files: models, links, joints, sensors Building a simple world and static objects in SDF; resource paths
4 Robot modeling for simulation (URDF ↔ SDF) Converting URDF/Xacro robots to SDF; visualizing in Gazebo Sim and RViz2
5 Dynamics and physics engines in Gazebo Tuning physics parameters, contact properties, and constraints for robots
6 Sensors in simulation: cameras, LiDAR, IMU, GPS Adding sensor plugins to robots; testing sensor topics and noise models
7 Gazebo plugins: world, model, sensor, system Writing a simple C++ model plugin; controlling a robot in simulation
8 ROS 2–Gazebo integration concepts (ros_gz_bridge, sim interfaces) Bridging Gazebo topics/services/actions to ROS 2; teleop of a sim robot
9 Building custom robots for Gazebo Sim Full pipeline: design URDF/Xacro → export SDF → spawn into custom world
10 Building custom worlds and environments Designing structured test arenas; importing meshes and assets
11 Advanced GUI and server configuration; logging and datasets Configuring GUI panels; logging world state and sensor data; replay
12 Simulated navigation: Nav2 in Gazebo Bringing up Nav2 on a Gazebo robot; maps, costmaps, navigation scenarios
13 Simulated manipulation: arms, grippers, contact-rich tasks Simulating a manipulator with Gazebo + MoveIt 2; grasping experiments
14 Multi-robot simulation and scalability Multiple robots in one world; namespaces, performance tuning, headless runs
15 From sim to real: fidelity, validation, and limits Comparing sim vs real logs; calibrating models; scenario stress testing
16 Final project presentations System demos: custom world, robot, plugins, and ROS 2 integration
