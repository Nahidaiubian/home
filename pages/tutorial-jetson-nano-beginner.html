Jetson Orin Nano Beginner Tutorial

Sixteen weeks, 32 class sessions (two per week): concepts in Class 1, hands-on work in Class 2. Each topic links to a detail page with notes and an embedded PDF viewer. For each session, save your handout as lecture.pdf in the matching folder assets/Tutorial/JetsonOrinNano/WxCy (for example W1C1 for week 1, class 1; W1C2 for week 1, class 2).

Schedule (16 weeks · 32 classes)

Week Class 1 (Concepts) Class 2 (Hands-on / Deep dive)
1 Course intro; what Jetson Orin Nano Super is; board overview; use cases Unboxing, hardware tour, SD card flashing, first boot, basic setup
2 JetPack, Ubuntu on Jetson, software stack overview System update, package installation, terminal basics, file system navigation
3 Linux essentials for beginners Working with files, permissions, editors, SSH, remote access
4 Python basics on Jetson Writing and running Python scripts; virtual environments; pip packages
5 Performance and power modes on Jetson Using nvpmodel, jetson_clocks, monitoring CPU/GPU/RAM/temperature
6 GPIO and hardware interfacing basics Controlling LED, reading buttons, simple GPIO projects
7 Cameras on Jetson: CSI vs USB, image pipeline basics Connecting a camera, testing video input, capturing images and video
8 OpenCV fundamentals for edge devices Image read/display/save, drawing, resizing, color spaces, webcam streaming
9 Computer vision basics Thresholding, contours, edge detection, simple object tracking
10 Deep learning on Jetson: PyTorch and pretrained models Installing/running PyTorch; image classification with a pretrained network
11 Object detection concepts Running a pretrained detection model on camera input in real time
12 Model optimization and TensorRT basics Converting and benchmarking an inference model for faster execution
13 Building small edge AI applications Project: smart camera, people detector, or simple alert system
14 Audio, sensors, and multimodal edge projects Microphone or sensor integration; combining sensor input with AI output
15 Deployment and reliability for Jetson projects Autostart apps, Docker basics, remote monitoring, troubleshooting
16 Final beginner projects and demos Project presentations, optimization review, next steps toward robotics/ROS 2
